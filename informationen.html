Stream Fragezeichen

Informationen zur Nutzung der Webseite


Einstellungen

Um mit der Nutzung der Webseite zu beginnen, muss zunächst ein Streaming Anbieter ausgewählt werden. Somit kann die Webseite automatisch auf die angeklickte Folge auf dem ausgewählten Anbieter weiterleiten.

Zusätzlich kann in den Einstellungen auch noch ausgewählt werden, ob das Theme (Hell oder Dunkel) vom Browser übernommen oder das jeweilig andere Theme verwendet werden soll. Ebenfalls kann man wählen, ob in der Folgenliste zuerst Folge 1 oder die neuste Folge angezeigt werden soll und ob die Folgen nach dem Erscheinungsdatum oder nach den Folgenarten (Spezialfälle, Adventsfolgen, Hörbücher) sortiert werden sollen.

Ausserdem kann eingestellt werden, ob die Folgentitel jeweils in der Liste (unterhalb vom Cover) angezeigt werden sollen und wie gross die Cover sein sollen. Der letzte Button ermöglicht den Zugriff auf die Folgen, welche im Zufallsgenerator ignoriert werden (siehe Zufällige Folge).

Folgeninformationen

Um die Folgeninfos anzuzeigen, muss auf den Info-Button der gewünschten Folge geklickt werden. Dieser Button wird auf Geräten mit Maus erst sichtbar, wenn mit dieser über das jeweilige Cover gefahren wird.

In der Folgeninfo ist der Klappentext, sowie der Autor, das Erscheinungsdatum, die Dauer und das Datum, an welchem die Folge zuletzt gehört wurde, ersichtlich. Das zuletzt gehörte Datum kann über das Stift-Icon direkt daneben bearbeitet werden. Sobald das vollständige Datum eingegeben wurde, kann es mit dem Bestätigungsbutton gespeichert werden.

Über das Cover kann die Folge direkt abgespielt werden. Zusätzlich kann hier auch ausgewählt werden, ob die Folge im Zufallsgenerator ignoriert werden soll und ob die Folge in die Watch List aufgenommen werden soll (siehe Zufällige Folge, Watch List).

Watch List

In der Watch List können Folgen abgespeichert werden, um später auf diese zugreifen zu können. Geöffnet wird die Watch List über das Icon in der Navigationsleiste. Die Folgen können entweder direkt in der Folgenliste über das Plus-Icon oder in der Folgeninfo hinzugefügt und wieder entfernt werden.

Verlauf

Im Verlauf werden sämtliche gehörte Folgen, nach der Reihenfolge des letzten Hörens, angezeigt. Geöffnet wird der Verlauf über das Icon in der Navigationsleiste. Wenn Folgen manuell hinzugefügt werden sollen, kann das in der Folgeninfo getan werden.

Secret: Wenn der 01.01.1999 als zuletzt gehört eingetragen wird, wird die Folge aus dem Verlauf entfernt.

Zufällige Folge

Über das Icon in der Navigationsleiste wird eine zufällige Folge vorgeschlagen. Falls diese nicht passen sollte, kann das Icon nochmals angeklickt werden, wodurch eine neue Folge angezeigt wird. Der Zufallsgenerator wählt immer eine der 20 am längsten nicht gehörten Fällen aus, wobei jede Folge nur einmal ausgewählt wird, bis die Webseite neu geladen wird. Falls die Folge gar nicht mehr vorgeschlagen werden soll, kann diese in der Folgeinfo zu den ignorierten Folgen hinzugefügt werden.

Alle ignorierten Folgen können in den Einstellungen angezeigt werden.

Advanced: Um zu verändern, aus wie vielen der am längsten nicht gehörten Folgen ausgewählt werden soll, muss auf dem Computer die Entwicklerkonsole des Browsers geöffnet werden. Hier kann folgender Befehl eingegeben werden: «random_settings[0] = 20». Die 20 kann durch die gewünschte Anzahl ersetzt werden.

Advanced: Um zu verändern, welche Folgenarten im Zufallsgenerator angezeigt werden sollen, muss auf dem Computer die Entwicklerkonsole des Browsers öffnet werden. Hier können die Befehle aus der folgenden Liste eingegeben werden. Wenn die Variable auf true gesetzt wird, wird die Folgenart angezeigt und wenn sie auf false gesetzt wird, wird sie ausgeblendet.

  • Normale Folgen: random_settings[1] = true
  • Special und Top Secret Folgen: random_settings[2] = true
  • Adventskalender: random_settings[3] = false
  • Kopfhörer-Hörspiele: random_settings[4] = false
  • Kurzgeschichten: random_settings[5] = false
  • Hörspiele zu Filmen: random_settings[6] = false
  • Liveshows: random_settings[7] = false
  • Hörbucher: random_settings[8] = false
  • Die Welt der drei Fragezeichen: random_settings[9] = false
Suche

Mit der Suche kann nach der Folgennummer, dem Folgenname, dem Folgenautor und nach der Folgenart gesucht werden. Geöffnet wird die Suche über das Icon in der Navigationsleiste.

Nutzerdaten

Über das Icon in der Navigationsleiste können die Nutzerdaten geöffnet werden. Hier können sämtliche Einstellungen von der Webseite exportiert, importiert oder zurückgesetzt werden. Ausserdem kann hier eine ID erstellt werden. Über diese ID können die Watch List, der Verlauf und die ignorierten Folgen über mehrere Geräte synchronisiert werden. Dazu wird diese neu erstellt und in das Textfeld eines anderen Gerätes eingefügt. Falls diese auf dem Gerät dann nicht mehr benötigt wird, kann man sich von dieser trennen oder die gesamte ID löschen.

Beachte, dass immer nur ein Gerät mit derselben ID aktiv sein sollte, da die Synchronisierung nicht in Echtzeit läuft, sondern erst wenn die Webseite neu geladen wird. Ansonsten kann es dazu führen, dass nur die Daten von einem Gerät gespeichert werden.

Falls Synchronisierungsprobleme auftreten sollten, kann bei einer Kontaktaufnahme auf fünf Tage ältere Versionen zugegriffen werden. Zur Bearbeitung benötigen wir die zugehörige ID und die ungefähre Zeit, bei der das Problem erkannt wurde. Ausserdem sollte danach die ID nicht mehr genutzt werden, bis das Problem behoben wurde. Um die Webseite weiter zu nutzen, kann man sich im Account Menü von der ID trennen. Bitte beachte, dass wir nicht jeden Tag die Benachrichtigungen überprüfen und somit ein Wiederherstellen nicht garantiert werden kann.
